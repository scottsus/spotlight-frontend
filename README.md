# Spotlight

## Usage

1. Clone

```
git clone git@github.com:susantoscott/spotlight.git
```

2. Build Dependencies

```
npm install
```

3. Rebuild the Extension

```
npm start
```

4. Apply the Extension

```
chrome://extensions => Load Unpacked => spotlight/dist
```

5. Enjoy!
